JavaScript es un lenguaje de programación interpretado que se utiliza
fundamentalmente para dotar de comportamiento dinámico a las páginas web.
Fue diseñado para que su código se incrustase en documentos HTML y fuera
interpretado por el navegador web.

Mediante JavaScript nos encargaremos de realizar acciones en el cliente,
como pueden ser pedir datos, confirmaciones, validación de datos en
formularios, incorporar efectos como texto que aparece y desaparece,
animaciones, acciones que se activan al pulsar botones, manejar ventanas
con mensajes de aviso al usuario…
